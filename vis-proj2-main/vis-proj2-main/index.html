The Truth Is Out There

Developers: Alexandru Dediu, Andrew Gerstenslager, Nick Murray, Zachary Carlson

Learn about reported UFO sightings from around the world!

Map: UFO sightings are shown on the map based on the location of the sighting. The points can be colored based on one of various attributes which can be chosen using the dropdown menu above the map. The layer icon in the upper-right corner provides multiple options for the map baselayer. Click and drag on the map to pan the map. Zooming with scroll is activated after clicking on the map, and clicking outside of the map disables that. Zooming can also be achieved with the plus and minus button in the upper-left corner. Hover the cursor over any point to see more information about that sighting.

Word Cloud: Each UFO sighting includes a text description, and the word cloud visualizes the frequency of certain words in the descriptions. Clicking on a word in the cloud will update the other visualizations on the page only to include sightings which contain that word in the description. A specific word can also be searched using the search bar. The reset button will reset all the visualizations to the default dataset.

Bar Charts: The bar charts on the page visualize the frequency of sightings grouped by different attributes. Hover over a bar to see more detailed information. Click and drag on any bar chart to create a selection. The other bar charts and map will update to reflect the data chosen withing the selection.

Data Source: National UFO Reporting Center Source Code: GitHub
